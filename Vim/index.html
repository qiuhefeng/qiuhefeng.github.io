Vim

发表于 2020-07-16 更新于 2020-07-20 阅读次数： Valine：

Vim是从vi演变过来的编辑器，支持可扩展。它的优点之一是可以脱离鼠标使用键盘进行高效的编辑，同时也有很多好用的插件用来提高效率。虽然它的入门比较难，但是随着使用时间增加，会越来越体会到它的高效和方便。

本文主要介绍Vim的配置和常用插件介绍，不涉及Vim的使用。

安装

各系统的安装方式可查看官方仓库的README文档。 官方仓库
Windows系统，不建议中中文官网下载安装，推荐官方仓库。Win官方仓库
MacOS系统，可以安装macvim，使用brew install macvim命令。

常用配置

进入Vim或MacVim后在命令模式下输入:e $MYVIMRC后编辑Vim的配置文件。配置文件在用户根目录下.vimrc。
修改配置文件后，需要source $MYVIMRC才能配置生效。
